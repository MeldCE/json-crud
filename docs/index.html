doCreate([replace], data)

Inserts data into the JSON database. Data can either be given as key-value parameter pairs, OR if the key field has been specified Object values. This function will fail if data was a given key already exists unless a Boolean is given for the first parameter, in which case, any data for an existing key will be replaced.

Parameters

  • [Boolean] replace (default false) :

    If true, if data already exists for a given key, that data will be replaced with the given data

  • ...Any data :

    Either key-value pairs, if the key field has been specified, Object values containing the key value, or a single Array containing one of the previous two options

Returns

Promise :

A promise that will resolve with an array containing keys of the inserted data.

  • [Boolean] expectSingle :
